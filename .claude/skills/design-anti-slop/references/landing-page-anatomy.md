## The 5-Section Structure

```
┌─────────────────────────┐
│       1. HERO           │  Above the fold
│   Value prop + CTA      │  One sentence, one button
├─────────────────────────┤
│    2. FEATURE SHOWCASE   │  3 features max
│   Benefits, not specs    │  Real screenshots
├─────────────────────────┤
│   3. SOCIAL PROOF        │  Quotes, logos, numbers
│   Real names + faces     │  Or before/after
├─────────────────────────┤
│    4. HOW IT WORKS       │  3 numbered steps
│   Action verbs           │  Visual flow
├─────────────────────────┤
│   5. CTA + FOOTER        │  Repeat primary CTA
│   Minimal footer         │  No clutter
└─────────────────────────┘
```

## Section Details

### 1. Hero

**Purpose**: Communicate the value proposition in < 5 seconds.

```
Good: "Ship 2x faster with AI code review"
Bad:  "Revolutionize your development workflow with our cutting-edge AI-powered platform"
```

Rules:
- One headline (6-12 words)
- One subheadline (optional, one sentence max)
- One CTA button with action verb ("Start free trial", not "Learn more")
- Product screenshot or short demo loop (not stock photo)
- Social proof micro-element: "Trusted by 500+ teams" or a logo row

### 2. Feature Showcase

**Purpose**: Show the top 3 reasons to care.

Rules:
- **3 features** — not 6, not 4, exactly 3
- Each with: icon/screenshot + benefit headline + one-line description
- Headlines are outcomes: "Save 4 hours/week" not "Automated workflows"
- Screenshots show the actual feature in use

### 3. Social Proof

**Purpose**: Reduce purchase anxiety with evidence.

Formats (pick one):
- **Testimonial cards** — real name, real photo, real title, specific quote
- **Logo wall** — 6-12 recognizable company logos, grayscale
- **Stats** — "10,000+ users", "99.9% uptime", "4.9/5 rating"
- **Before/After** — visual comparison of with/without the product

### 4. How It Works

**Purpose**: Reduce friction by showing simplicity.

Rules:
- **3 steps** — numbered, not bulleted
- Each step has an **action verb**: "Connect", "Configure", "Ship"
- Visual flow: step 1 → step 2 → step 3 (arrows or timeline)
- Optional: small screenshot per step

### 5. CTA + Footer

**Purpose**: Convert the convinced reader.

Rules:
- **Repeat the primary CTA** — same button text as hero
- Optional: add a secondary CTA ("Book a demo" next to "Start free trial")
- Footer: navigation links, legal, social icons
- **No newsletter signup** unless it's the primary conversion action

## Common Mistakes per Section

| Section | Mistake | Fix |
|---------|---------|-----|
| Hero | Paragraph-length subtitle | One sentence max |
| Hero | 3 CTA buttons | One primary, one secondary max |
| Hero | Abstract illustration | Product screenshot |
| Features | 6+ features | Cut to top 3 |
| Features | Feature names as headlines | Benefit outcomes as headlines |
| Social Proof | Fake testimonials | Use real or remove section |
| Social Proof | Unknown company logos | Either recognizable logos or none |
| How It Works | 5+ steps | Collapse to 3 |
| How It Works | Passive voice | Active verb per step |
| CTA | Different CTA than hero | Repeat hero CTA exactly |
| Footer | Newsletter + social + blog + resources | Minimal: nav + legal |

## Mobile Considerations

- Hero: stack vertically, CTA full-width
- Features: single column, swipeable if needed
- Social proof: horizontal scroll for logos, stacked for testimonials
- How it works: vertical timeline instead of horizontal
- CTA: sticky bottom bar on mobile
